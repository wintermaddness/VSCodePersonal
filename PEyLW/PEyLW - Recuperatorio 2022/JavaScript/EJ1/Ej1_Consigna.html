Recuperatorio JS Ejercicio 1

  • Consigna
  • HOJA Ejercicio

Existen 3 bases de datos con números llamadas A1, A2 y A3. Las encontrará en el archivo de resolución. Se desea saber, dado un número, en cuantas bases de datos aparece.

  • El operador ingresa un número
  • El sistema debe controlar que el número sea válido y dentro del rango 0..2000
  • El sistema debe informar en cuantas bases de datos el número fue encontrado. Span ResultadosBusqueda
  • Opcional: Muestrar en que bases de datos se encuentran los números. Span ResultadosDBsEncontradas

Importante: No se toma en consideración cuestiones de performance

Hay numeros que están repetidos en las bases de datos. Los mismos deben ser contados por UNICA vez. Se considerará mal el ejercicio que cuente los números repetidos.

Completar la página Ej1_Consigna.html con el código Javascript que corresponde

NO UTILICE ARCHIVOS JAVASCRIPT EXTERNOS.
